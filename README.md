# Searches and Sorts

## Searches

| Searches Implemented      | Description    |
| ------------------------- | -------------- |
| Linear Search             | O{n}           |
| Binary Search             | O{log(n)}      |
| Jump Search               | O{(n/m)+(m-1)} |

## Sorts

| Sorts Implemented         | Description |
| ------------------------- | ----------- |
| Bubble Sort               | O{n^2}      |
| Selection Sort            | O{n^2}      |
|                           |             |
